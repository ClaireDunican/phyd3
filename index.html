Welcome to PhyD3!


PhyD3 is a phylogenetic tree viewer based on d3.js.
It was developed as an alternative to Archaeopteryx inspired by d3.phylogram.js.

You can use it, host it, embed it, extend it. Licensed under GPL v3.
See more info in the documentation.
